== Description ==

Adds Windows Azure End User Analytics to WordPress pages.

== Installation ==

This section describes how to install this plugin and get it working.

1. Upload the windows-azure-end-user-analytics-unofficial folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Settings -> Windows Azure End User Analytics (Unofficial) Settings screen.
4. Enter the AppInsights App Key from your Windows Azure Website.
5. Click "Save Changes".
6. Windows Azure End User Analytics should be enabled and start collecting usage analytic data. Usage analytic information will begin to appear after about 5-10 minutes.
